# rjb2022
**软件杯2022**

## 环境
- Anaconda
- Python 3.7.x
    - Django
    - PaddleRS
- Nodejs (开发)

## 语言
- Python
- TypeScript
- HTML, CSS

采用原生HTML + TypeScript，无框架

## 启动服务

```
$ cd ./server
$ python ./manage.py run
```
如果顺利，前端网址在 http://127.0.0.1:8000/static/index.html 已打开 

后端调试地址在 http://127.0.0.1:8000/post_test.html

## 开发环境自动编译TS
```
$ cd ./client
$ npm run wpw
```
需要vscode live server插件
